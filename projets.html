Développement d'une application de gestion de comptes bancaires avec IHM

Projet d'équipe mené en 2023 de 3 semaines consistant à développer une application bancaire en Java.

Description

En équipe de 3, nous avons développé une interface IHM pour la gestion de compte bancaire en Java et JavaFX. Ce fut notre premier projet utilisant GitHub. L'application permet la gestion d'agences bancaires, de comptes de clients, de virements entre compte et de simulation de remboursement d'emprunt en fonction du taux voulu.

Tâches effectuées

- Développement d'une application en Java
- Utilisation de GitHub, Eclipse
- Planification et répartition des tâches au sein de l’équipe
- Création d’une documentation technique en markdown
